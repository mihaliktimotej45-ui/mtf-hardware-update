// FRAMEWORK 01

NUDGE THEORY

Richard Thaler & Cass Sunstein (2008) definujú "nudge" ako jemné úpravy prostredia, ktoré vedú ľudí k lepším rozhodnutiam bez obmedzenia slobody voľby.

> Nobel Prize Economics 2017
> Aplikácia: Architektura výberu
B
// FRAMEWORK 02

BROKEN WINDOWS

Wilson & Kelling (1982): Prostredie vysiela signály, ktoré ovplyvňujú správanie. Upravené prostredie = lepšie rozhodnutia. Zaprataný kantin = nezdravý výber.

> Vizuálne podnety = zmena správania
> Prostredie ako architekt
